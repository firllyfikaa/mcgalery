MC Galery

Proyek ini menyediakan galeri Minecraft berbasis web. Aplikasi menampilkan screenshot, struktur base, progres dunia, dan dokumentasi server.

Fitur
Tampilan galeri yang cepat.
Navigasi sederhana.
Komponen UI siap pakai.
Dukungan dark mode.
Optimasi gambar otomatis.
Struktur kode mudah dikembangkan.




Cara menjalankan
npm install
npm run dev
npm run build
npm start

Akses http://localhost:3000.
